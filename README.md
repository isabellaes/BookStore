# Webbshop - (under construction)

Webbshop - Simple webbshop demo for portfolio.

## Installation

Clone repository

```sh
git clone https://github.com/isabellaes/Webbshop
```

Frontend

```sh
cd client
npm install
npm start
```

Backend . Not implemented

```sh
cd server
dotnet run
```

- Typescript
- Sass
